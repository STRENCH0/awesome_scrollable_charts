# Scrollable Charts Example

This is an example app demonstrating the features of the `scrollable_charts` package.

## Features Demonstrated

- **LineChart**: Interactive line chart with smooth scrolling and animations
- **StackedAreaChart**: Cumulative area chart showing multiple data series
- Customizable styling options (colors, markers, labels)
- Smooth animations and scroll physics
- Data labels and tooltips
- Grid lines and axis styling

## Running the Example

To run this example app:

```bash
flutter run
```

## Code Structure

- `lib/main.dart` - Main app with chart examples
- `lib/sample_data.dart` - Sample data generators for demonstration
